So Offal It's Good

In my second week of learning JavaScript, I had lots of fun working with the Yummly API to build a web app that searches for offal recipes—dishes that include the organs, entrails, or extremities of animals. The idea came from the meals that regularly come out of my mother’s kitchen, and the meals of my parents’ childhood. The interface is designed to introduce people to the concept of offal and make it simple for beginners to find accessible and delicious recipes.

Featured Skills:

JavaScript, Working with REST APIs, AJAX, JSON, jQuery, Web Design, CSS Animation, Responsive Design

Responsive Layout

For my Week 3 Project at HackerYou, I was given a PSD file to develop into a responsive single-page website. The layout is built on the grid system, and I loved using Sass variables and mixins to streamline the design process. A personal highlight from this project was experimenting with scroll-activated animations and mobile-friendly navigation bars.

Stkr jQuery Plugin

This jQuery plugin makes it easy to manipulate when, where, and how an element becomes fixed on a webpage. You can identify a designated “sticky zone” for your fixed elements, have them fade in/out of view, or appear seemingly out of nowhere. And as a bonus feature, you can even set the sticky effects to be triggered by a horizontal scroll!
